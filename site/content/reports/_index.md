  ### 謝辞 / Acknowledgements
  本センターの支援を受けた成果を報告・出版される際は、謝辞（Acknowledgements）に本センターの名称を記載いただけますと幸いです。規定のフォーマットはございませんが、以下のような記述をご検討ください。

  If you publish results supported by this center, we would appreciate it if you could include an acknowledgement. While there is no fixed format, please consider the following examples:

  - **日本語表記例**: 本研究は、九州大学汎オミクス計測・計算科学センターの共同研究支援を受けたものです。
  - **English Example**: This work was supported by the Pan-Omics Data-Driven Innovation Research Center, Kyushu University.
