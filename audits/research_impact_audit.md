# research_impact_audit.md

## Task

Evaluate the research impact potential of the laboratory website.

The audit should determine how effectively the site enables dissemination and reuse of research outputs.

Focus on research visibility beyond publications.

---

# Scope

Analyze whether the site exposes research artifacts that enable:

citation
reuse
collaboration
media visibility

---

# 1 Publication Accessibility

Check whether publications include:

PDF links
DOI links
arXiv links

Flag issues:

publications listed without access links.

---

# 2 Code Availability

Check whether research papers link to code repositories.

Examples:

GitHub
GitLab

Flag if research appears computational but no code is available.

---

# 3 Dataset Availability

Check if datasets are shared.

Possible dataset hosting:

Zenodo
Figshare
institution repositories

Flag missing datasets for data-driven research.

---

# 4 Preprint Visibility

Check whether the site links to:

arXiv
institution preprint repositories

This increases citation speed.

---

# 5 Research Artifact Integration

Evaluate whether publications connect to:

paper
code
dataset
slides

Ideal structure:

Paper
Code
Dataset
Slides

Flag if these artifacts are disconnected.

---

# 6 Research Topic Pages

Check if research topics highlight:

representative papers
key figures
impact statement

Flag if research pages only list titles.

---

# 7 External Visibility

Check for links to:

Google Scholar
ORCID
GitHub profile

These improve discoverability.

---

# 8 Knowledge Graph Potential

Check whether the site structure supports linking between:

projects
papers
datasets
software

Flag if pages are isolated.

---

# 9 Citation Amplification

Evaluate if the site supports citation amplification through:

blog posts
research highlights
press mentions

---

# 10 Research Timeline

Check whether the site shows research evolution:

timeline
project history
major milestones

---

# 11 Collaboration Signals

Check whether the site includes:

collaboration invitations
industry partnerships
grant projects

---

# 12 Impact Metrics

Check whether the site shows:

citation counts
downloads
GitHub stars
dataset usage

These increase perceived impact.

---

# 13 Report Output

Codex should generate:

impact readiness score
strengths
weaknesses
missing research artifacts
recommended improvements

---

# Priority Levels

critical
recommended
optional

---

# Final Goal

Transform the laboratory website into a research dissemination platform rather than a simple publication list.
